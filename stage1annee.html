Missions

Développement d'une IHM pour le lancement de scripts

→  Création d'une interface web permettant à l'utilisateur de saisir des données, sélectionner des fichiers d'entrée, choisir un script à exécuter, et obtenir des fichiers de sortie.

→  Intégration des fonctionnalités de gestion des erreurs, avec affichage des messages d'erreur retournés par les scripts.

Webscraping : script Python pour rechercher des informations de contact

→  Développement d'un script qui analyse une page web pour extraire des informations de contact comme les numéros de téléphone, adresses e-mail, adresses postales et noms de personnes.

Webscraping : script Python pour rechercher des pages web

→  Élaboration d'un script capable de rechercher des pages web en utilisant des mots spécifiques dans la recherche (par exemple, sur Google), et d'extraire les premières URLs pertinentes.

Outils utilisés

IDE

Tout au long du stage, l'IDE VS Code m'a permis de travailler efficacement sur toutes les missions grâce à ses nombreuses extensions et à la gestion optimale des fichiers.

Frontend

J'ai utilisé HTML/CSS pour structurer et designer l'interface Homme/Machine et JavaScript pour la rendre interactive.

Backend

J'ai utilisé Python pour développer les scripts permettants l'extraction de données du web. Grâce à ses nombreuses librairies, le webscraping a été facilité.

Difficultés Rencontrées




Extraction des adresses avec des patterns

Développer des expressions régulières pour extraire avec précision les adresses postales à partir de différentes pages web a été un vrai challenge étant donné la variabilité des formats. Après plusieurs essais, j'ai pu affiner les patterns regex et capturer les informations nécessaires.




Exécution de scripts Python depuis du HTML

L'objectif initial était de lancer des scripts Python à partir de l'interface, plutôt que du JavaScript. Cependant, cette intégration s'est avérée complexe en raison des restrictions des navigateurs et des limitations techniques. Malgré plusieurs tentatives et recherches, je n'ai pas réussi à contourner ces obstacles.
