Missions

Développement d'une IHM pour le lancement de scripts

→  Création d'une interface web permettant à l'utilisateur de saisir des données, sélectionner des fichiers d'entrée, choisir un script à exécuter, et obtenir des fichiers de sortie.

→  Intégration des fonctionnalités de gestion des erreurs, avec affichage des messages d'erreur retournés par les scripts.

Webscraping : script Python pour rechercher des informations de contact

→  Développement d'un script qui analyse une page web pour extraire des informations de contact comme les numéros de téléphone, adresses e-mail, adresses postales et noms de personnes.

Webscraping : script Python pour rechercher des pages web

→  Élaboration d'un script capable de rechercher des pages web en utilisant des mots spécifiques dans la recherche (par exemple, sur Google), et d'extraire les premières URLs pertinentes.

Récupération des données depuis l'API

La récupération des données depuis l'API a été un défi, principalement parce qu'il s'agissait de manipuler des données externes qui ne sont pas stockées dans notre propre base de données.




Gestion des favoris

La gestion des favoris a présenté une complexité supplémentaire, car il fallait synchroniser les choix des utilisateurs entre la carte et leur espace personnel, tout en enregistrant ces informations dans la base de données.
